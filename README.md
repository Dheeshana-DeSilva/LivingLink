# LivingLink

LivingLink is a microservice-based roommate and shared accommodation matching platform.

## Main Features

- User registration and login
- Lifestyle profile creation
- Accommodation listing management
- Roommate preference management
- Compatibility-based matching
- Visit scheduling
- Notifications
- Admin moderation

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Redux Toolkit
- Axios
- React Router

### Backend
- Spring Boot Microservices
- Spring Cloud Gateway
- Eureka Service Discovery
- Spring Security + JWT

### Database
- PostgreSQL

### Deployment
- Docker Compose
